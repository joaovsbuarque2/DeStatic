# Blog

## Creating a Post

```bash
hugo new content blog/your-post-title.md
```

## Front Matter

```yaml
---
title: "Your Post Title"
description: "Brief description for search results and cards"
date: 2025-01-01
author: "Your Name"
categories: ["Category1", "Category2"]
tags: ["tag1", "tag2"]
image: "/images/blog/your-image.jpg"
draft: false
---
```

## Categories & Tags

Categories and tags are automatically generated. View all posts in a category:

```
your-site.com/categories/category-name/
your-site.com/tags/tag-name/
```
